## 🚀 Why Throughthink?

Throughthink is a sleek, minimalist, and powerful AI chat client designed for users who want direct access to the latest LLMs without the constraints of traditional subscription services.

By connecting directly to **OpenRouter**, Throughthink offers:

*   **💰 Pay-Per-Use:** Stop paying $20/month for services you barely use. With Throughthink, you pay only for the tokens you consume. Most users spend less than $2/month.
*   **🚫 No Rate Limits:** Never get locked out of your chat because you sent "too many messages."
*   **🧠 Model Agnostic:** Switch instantly between **Claude 3.5 Sonnet**, **GPT-4o**, **DeepSeek R1**, and hundreds of other models without needing multiple subscriptions.
*   **⚡ High Performance:** A lightweight, responsive interface built for speed and focus.

## ✨ Key Features

*   **Thinking Animation:** Real-time visual feedback for models that support internal reasoning (e.g., DeepSeek R1).
*   **Collapsible Thought Process:** Easily toggle the model's reasoning steps to keep your chat clean.
*   **Custom Model Support:** Add any model ID from OpenRouter's vast library directly in the settings.
*   **Sleek Aesthetic:** A dark, refined interface with subtle flowing backgrounds and a distinctive design language.
*   **Tutorial Built-in:** New to OpenRouter? Our interactive tutorial guides you through getting your API key and finding model IDs.

## 🛠 Getting Started

1.  **Get an OpenRouter Key:**
    *   Sign up at [OpenRouter.ai](https://openrouter.ai).
    *   Add a small amount of credit to your account.
    *   Generate an API Key (starts with `sk-or-v1-`).
2.  **Configure Throughthink:**
    *   Open the **Settings** menu (gear icon).
    *   Paste your API Key into the designated field.
3.  **Start Chatting:**
    *   Select your preferred model from the dropdown and start your first conversation!

---

## ⚙️ Custom Models

Want to use a specific model not in our default list?
1.  Browse [OpenRouter Models](https://openrouter.ai/models).
2.  Find the API ID of the model you want (e.g., `anthropic/claude-3.5-sonnet`).
3.  Paste that ID into the **Add Custom Model ID** field in Settings and click **+**.
